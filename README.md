# TaskFlow

Gerenciador de tarefas estilo Kanban (Trello), desenvolvido por **Israel Alcantara**.

## Funcionalidades

- 3 colunas: A Fazer, Em Progresso, Concluído
- Arrastar e soltar (drag & drop) entre colunas
- Criar, editar e excluir tarefas
- Prioridades (Alta, Média, Baixa) e datas de entrega
- Persistência local (localStorage)
- Interface responsiva em português

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- @dnd-kit (drag and drop)
- shadcn/ui

## Como rodar

```bash
npm install
npm run dev
```
